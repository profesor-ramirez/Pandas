1. Introducción a Pandas y Estructuras Principales

Pandas es la herramienta más popular en Python para el análisis de datos. Introduce dos estructuras de datos principales que facilitan mucho el trabajo: Series y DataFrames.

Series

Una Serie es un arreglo unidimensional (como una columna de una hoja de cálculo) capaz de almacenar cualquier tipo de dato (enteros, strings, flotantes, objetos de Python, etc.). Tiene un índice que etiqueta cada elemento.

# Primero, importamos la librería
import pandas as pd
import numpy as np # Numpy se usa a menudo con Pandas

# Creación de una Serie desde una lista
s = pd.Series([1, 3, 5, np.nan, 6, 8])

print(s)

Salida esperada:

0    1.0
1    3.0
2    5.0
3    NaN
4    6.0
5    8.0
dtype: float64

DataFrame

Un DataFrame es una estructura de datos tabular bidimensional (como una hoja de cálculo completa o una tabla de SQL) con columnas que pueden ser de diferentes tipos. Es la estructura de Pandas más utilizada.

# Creación de un DataFrame desde un diccionario
data = {
    'Nombre': ['Ana', 'Luis', 'Carlos', 'Marta'],
    'Edad': [28, 34, 29, 42],
    'Ciudad': ['Madrid', 'Barcelona', 'Madrid', 'Valencia']
}
df = pd.DataFrame(data)

print(df)

Salida esperada:

   Nombre  Edad     Ciudad
0     Ana    28     Madrid
1    Luis    34  Barcelona
2  Carlos    29     Madrid
3   Marta    42   Valencia

3. Selección, Filtrado y Limpieza de Datos

Selección de Columnas

# Seleccionar una sola columna (devuelve una Serie)
nombres = df['Nombre']
print(nombres)

# Seleccionar múltiples columnas (devuelve un DataFrame)
info_personal = df[['Nombre', 'Edad']]
print(info_personal)

Salida (múltiples columnas):

   Nombre  Edad
0     Ana    28
1    Luis    34
2  Carlos    29
3   Marta    42

Filtrado de Filas (Basado en condiciones)

Esto se conoce como "indexación booleana".

# Filtrar personas mayores de 30 años
mayores_de_30 = df[df['Edad'] > 30]
print(mayores_de_30)

# Filtrar personas que viven en Madrid
de_madrid = df[df['Ciudad'] == 'Madrid']
print(de_madrid)

Salida (mayores de 30):

  Nombre  Edad     Ciudad
1   Luis    34  Barcelona
3  Marta    42   Valencia

Limpieza de Datos (Manejo de Nulos)

Los datos del mundo real rara vez están completos. np.nan representa datos faltantes.

# Creemos un DataFrame con datos faltantes
data_con_nulos = {
    'A': [1, 2, np.nan, 4],
    'B': [5, np.nan, np.nan, 8],
    'C': [9, 10, 11, 12]
}
df_nulos = pd.DataFrame(data_con_nulos)
print("Original:")
print(df_nulos)

# Opción 1: Eliminar filas con cualquier valor nulo
df_sin_nulos_filas = df_nulos.dropna()
print("\nDespués de dropna():")
print(df_sin_nulos_filas)

# Opción 2: Rellenar valores nulos (p.ej., con 0 o la media)
df_relleno = df_nulos.fillna(value=0)
print("\nDespués de fillna(0):")
print(df_relleno)

Salida (rellenando con 0):

Después de fillna(0):
     A    B   C
0  1.0  5.0   9
1  2.0  0.0  10
2  0.0  0.0  11
3  4.0  8.0  12

4. Agrupamiento, Unión y Resumen de Información

Agrupamiento (Group By)

La operación "Group By" es fundamental para el análisis. Implica uno o más de los siguientes pasos:

  • Dividir (Splitting): los datos en grupos basados en algún criterio.
  • Aplicar (Applying): una función a cada grupo independientemente.
  • Combinar (Combining): los resultados en una estructura de datos.

# Usando el primer DataFrame 'df'
# Agrupemos por 'Ciudad' y contemos cuántas personas hay en cada una
conteo_ciudad = df.groupby('Ciudad').size()
print("Conteo por ciudad:")
print(conteo_ciudad)

# Agrupemos por 'Ciudad' y calculemos la edad media
edad_media_ciudad = df.groupby('Ciudad')['Edad'].mean()
print("\nEdad media por ciudad:")
print(edad_media_ciudad)

Salida (Edad media):

Edad media por ciudad:
Ciudad
Barcelona    34.0
Madrid       28.5
Valencia     42.0
Name: Edad, dtype: float64

Unión (Merge/Join)

Similar a los JOINs de SQL, Pandas puede combinar DataFrames.

# Creemos un segundo DataFrame
data_profesion = {
    'Nombre': ['Ana', 'Luis', 'Carlos', 'Marta'],
    'Profesion': ['Ingeniera', 'Arquitecto', 'Doctor', 'Diseñadora']
}
df_profesion = pd.DataFrame(data_profesion)

# Unimos los dos DataFrames usando 'Nombre' como clave
df_completo = pd.merge(df, df_profesion, on='Nombre')
print(df_completo)

Salida esperada:

   Nombre  Edad     Ciudad    Profesion
0     Ana    28     Madrid    Ingeniera
1    Luis    34  Barcelona   Arquitecto
2  Carlos    29     Madrid       Doctor
3   Marta    42   Valencia   Diseñadora
